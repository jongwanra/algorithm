# Algorithm

### 알고리즘 문제 풀이 접근 방법

https://www.youtube.com/watch?v=Wx91sqfouCs

1. 문제 풀기 전에 **접근 방법**을 먼저 고민한다. (+ 상수 조건 확인)
2. 30분 내로 못풀면 정답을 보고 공부해서 흡수한다.
3. 코드를 다 작성 했으면 바로 제출하지 말고 아래를 수행한다.
    1. 일반 케이스, 엣지 케이스 1 ~ 2개 확인
    2. 시간 복잡도 / 공간 복잡도 분석
4. 풀고 난 이후, 다른 사람들의 다양한 풀이와 깔끔한 코드를 보고 공부한다.
5. 한 달 안에 다시 와서 푼다.  15분 내로 풀 수 있을 때 까지 반복한다.

- 시간복잡도
    - **입력의 크기**에 따라 얼마나 많은 연산을 수행하는지 나타낸다.
    - 자바의 경우 1초에 1억(10^8) 번 연산이 가능하다.
